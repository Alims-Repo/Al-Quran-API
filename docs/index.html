About the Quran API

Built for Android developers, the Quran API provides easy and efficient access to Quranic data, translations, and related resources, ideal for educational and religious applications.

How It Works

The Quran API uses optimized binary storage for rapid data access. Modular design enables flexible use, allowing developers to access only the features they need.

Get Started

Add the following dependency to your project‚Äôs build.gradle file:

implementation("com.nelu:quran-api:0.9.1-beta")

Configure GitHub Packages in `settings.gradle.kts`:

dependencyResolutionManagement {
repositoriesMode.set(RepositoriesMode.FAIL_ON_PROJECT_REPOS)
repositories {
    google()
    mavenCentral()
    maven {
        url = uri("https://maven.pkg.github.com/Alims-Repo/Al-Quran-API")
    }
}
}

Usage Examples

Retrieve Surahs

val surahList = QuranAPI.SURAH.getSurahList()
surahList.forEach { surah ->
    println("Surah ${surah.number}: ${surah.englishName}")
}

Download a Translation

val translationCode = "en_saheeh"
QuranAPI.TRANSLATION.downloadTranslation(translationCode, object : BaseTranslation.TranslationDownloadListener {
    override fun onSuccess() { println("Translation downloaded!") }
    override fun onFailure(e: Exception) { println("Download failed: ${e.message}") }
    override fun onProgress(type: BaseTranslation.TranslationDownloadListener.Type, progress: Int) {
        println("Progress: $progress%")
    }
})
